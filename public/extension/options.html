Options d'affichage

✓ Enregistré
Pour modifier le raccourci clavier, allez dans
about:addons → ⚙️ → Gérer les raccourcis d'extension

⌨️ Commandes au clavier

  • Ctrl+M pour activer/désactiver l'extension.
  • Entrée pour coloriser l'élément focusé. Si activable, un 2e Entrée l'active.
  • Tab / Maj.+Tab pour naviguer.
  • Échap. pour désactiver.

🖱️ Commandes à la souris

  • Cliquer sur l'icône pour activer. Cliquer sur un élément pour coloriser.
  • Si activable, un 2e clic l'active.
  • Échap. pour désactiver la coloration.
