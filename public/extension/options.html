Pour modifier le raccourci clavier, allez dans
about:addons → ⚙️ → Gérer les raccourcis d'extension

⌨️ Commandes au clavier

  • Ctrl+Maj+J pour activer/désactiver l'extension.
  • Tab pour naviguer entre les éléments.
  • Entrée pour coloriser l'élément focusé. Si activable, un 2e Entrée l'active.
  • Échap. pour revenir en mode attente.
  • + et - pour changer la portée (mot / phrase / paragraphe / tout).

🖱️ Commandes à la souris

  • Cliquer sur l'icône 🎨 dans la barre d'outils pour activer.
  • Cliquer ou survoler un élément pour le coloriser (selon le réglage).
  • Si activable, un 2e clic l'active.
  • Clic droit pour revenir en mode attente.
  • + et - pour changer la portée.
